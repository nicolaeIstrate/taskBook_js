The values of two integer variables A and B are given. If the values are not equal then assign the sum of given
values to each variable, otherwise assign zero value to each variable. Output the new values of the variables A and B.

Даны две переменные целого типа: A и B. Если их значения не равны, то присвоить каждой переменной сумму этих
значений, а если равны, то присвоить переменным нулевые значения. Вывести новые значения переменных A и B.
